---

最被低估的用法是**让组件用 CSS var 暴露"插槽"**,而不是 prop:

```css
.button {
  background: var(--button-bg, var(--accent));
  padding: var(--button-padding, 0.5em 1em);
}
```

外层只需 `style={{ "--button-bg": "red" }}` 就改了。比 React 的 className 注入或 prop drilling 干净得多。
