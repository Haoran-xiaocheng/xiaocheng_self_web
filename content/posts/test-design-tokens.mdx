我的判断:如果一个值**会出现在多个媒介**(网页 + iOS + Android + Figma),那它值得成为 token。否则就是 CSS 变量。

工具链选 Style Dictionary 或者 Tokens Studio。Figma 那一头要让设计师参与维护,不然永远在追赶。
